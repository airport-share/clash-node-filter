# 基于clash内核的节点筛选器

节点池的节点太多, 而且大部分不能用, 不好看

没关系, 通过这个节点筛选器就能筛选出能用的节点, 并备注节点的地区及ISP

# 使用方法

## 准备工作

- nodejs >= 10
- clash(clashMeta 等)

## 克隆仓库 or 下载源码 

### 克隆仓库
```bash
git clone git@github.com:airport-share/clash-node-filter.git
```
